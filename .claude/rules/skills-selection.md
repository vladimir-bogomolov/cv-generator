# Skills Selection Rules

Skills must come from:

data/professional_skills.md  
data/university_skills.md

---

# Selection Process

1. Extract keywords from the job analysis
2. Match them against candidate skills
3. Rank skills by relevance

Priority order:

1 direct match  
2 ecosystem match  
3 domain match  

Example:

Job requires:

React  
TypeScript  

Valid matches:

React  
TypeScript  
Next.js  
JavaScript  

---

# University Skills

University skills may be included only if relevant.

They must be labeled:

(university project experience)

Example:

Solidity (university project experience)
