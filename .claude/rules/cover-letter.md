# Cover Letter Rules

# Language

Write the cover letter in the same language as the job description. If the job posting is in Dutch, write in Dutch. If in English, write in English.

The cover letter must be short and natural.

Maximum length: 150 words.

Opening:

Start with "Hi," — never use "Dear X team" or similar formal openers.

Structure:

Paragraph 1
Introduce yourself and mention the role.

Paragraph 2  
Say that I am looking for a position where I can use my existing skills to be productive from the day 1 and the same time to learn something new on the job and expand my expertise. Emphasize the desire to grow and learn.

Paragraph 3  (optional)
Show interest in the company.

Tone:

professional  
natural  
direct

Avoid generic phrases.

Never use em dashes.

Keep writing simple and human-like, don't use typical phrases. Make it to stand out
